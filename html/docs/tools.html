pgRouting Tools¶

There are several tools available that can help to work with pgRouting. Most of the currently listed tools help with OpenStreetMap data.

Also take a look at the pgRouting Howto’s page.

  • OSM2PostGIS - Import OSM data to PostGIS
  • osm2pgrouting - Import OSM data into pgRouting Database
    • Features
    • Requirements
    • How to install
    • How to use

Links to other websites¶

  • osm2po - tool to convert OSM data into a routable format
  • osm4routing - OpenStreetMap data parser to turn them into a nodes-edges adapted for routing applications

Note

If you know about other tools and software that can be used to work with pgRouting, send an email to project@pgrouting.org and we will add it to the list.
