How to Contribute¶

Code Snippets¶

To share your code snippets with the community Github’s Gist is a useful tool.

How to contribute snippets?

  1. Make sure you snippet has all the pgRouting snippet header included and filled out
--
-- TITLE:
-- DESCRIPTION:
-- USE CASES:
-- EXAMPLE(S):
-- LICENSE:
-- AUTHOR:
-- EMAIL:
--
  1. Post your snippet to Gist: https://gist.github.com
  2. Create a ticket on the pgRouting issue tracker and add the label Snippet.
  3. Post an email to pgrouting-users@lists.osgeo.org with links to your Gist and some explanation.
