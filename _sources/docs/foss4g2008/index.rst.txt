Workplace Access
----------------

* Download the compressed VMware image from http://download.osgeo.org/pgrouting/foss4g2008/ and extract the file
* Start VM Player and load the workshop virtual machine
* If root access is required, use

.. code-block:: bash

	username: pgrouting
	password: foss4g
