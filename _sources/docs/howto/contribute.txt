======================
 How to Contribute
======================

Code Snippets
-------------

To share your code snippets with the community `Github's Gist <https://gist.github.com>`_ 
is a useful tool. 

**How to contribute snippets?**

1. Make sure you snippet has all the pgRouting snippet header included and filled out

.. code-block:: sql

  --
  -- TITLE:
  -- DESCRIPTION:
  -- USE CASES:
  -- EXAMPLE(S):
  -- LICENSE: 
  -- AUTHOR:
  -- EMAIL:
  --
  
2. Post your snippet to Gist: https://gist.github.com
3. Create a ticket on the `pgRouting issue tracker <https://github.com/pgrouting/pgrouting/issues>`_ and add the label ``Snippet``.
4. Post an email to pgrouting-users@lists.osgeo.org with links to your Gist and some explanation.
